# 00--04: CLI → ValidateStructure

## 개요

CLI에서 문서 구조 검증 명령을 실행합니다.

## 인터페이스 정의

### 입력 (CLI → ValidateStructure)

**명령어**: `mdoc validate structure [options]`

**Options 타입**:
```typescript
{
  project?: string;  // -p, --project
}
```

**처리**: `src/cli.ts:72-86`

### 출력 (ValidateStructure → CLI)

**StructureValidationResult 타입**:
```typescript
interface StructureValidationResult {
  success: boolean;
  totalFiles: number;
  errors: StructureError[];
  warnings: StructureWarning[];
}
```

**Exit Code**:
- `0`: 검증 성공
- `1`: 검증 실패 또는 에러
