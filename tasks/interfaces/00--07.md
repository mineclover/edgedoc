# 00--07: CLI → Sync

## 개요

CLI에서 코드 참조 동기화 명령을 실행합니다. (개발 예정)

## 인터페이스 정의

### 입력 (CLI → Sync)

**명령어**: `mdoc sync`

**Options 타입**: (아직 정의되지 않음)

**처리**: `src/cli.ts:158-170`

### 출력 (Sync → CLI)

**SyncResult 타입**:
```typescript
interface SyncResult {
  success: boolean;
  totalBlocks: number;
  updatedBlocks: number;
  failedBlocks: number;
}
```

**Exit Code**:
- `0`: 동기화 성공
- `1`: 동기화 실패 또는 에러
